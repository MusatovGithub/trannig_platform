# Источник стилей в локальной разработке

## Правило

Для локальной разработки и тестирования основным источником CSS/JS является директория `static/`.

## Практика изменений

- При изменении стилей сначала актуализировать файлы в `static/`.
- Если в репозитории используется дублирующая директория `static_cdn/`, поддерживать синхронность только как вторичный шаг.
- При расхождении между `static/` и `static_cdn/` в локальном контексте приоритет у `static/`.
